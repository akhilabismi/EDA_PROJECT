# Airbnb EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Airbnb Listings dataset.

## Objectives
- Understand dataset structure
- Identify missing values
- Detect data quality issues
- Prepare for cleaning and analysis

## Dataset
Airbnb Listings dataset from Kaggle.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
